Keyboard Shortcuts & Help

Basic workflow

  1. Select a sequence from the sidebar.
  2. Click a drone name in the Active drone bar to select it.
  3. Click and drag on the canvas to draw a bounding box.
  4. Click Save when done — changes are staged in a temp file until then.

Navigation

D / →Next frame
A / ←Previous frame
SpacePlay / Pause
Scroll wheelZoom in / out
Time barClick or drag to jump to any time

Editing

Drag on canvasDraw new box (requires active drone)
Drag box interiorMove box
Drag box cornerResize box
RErase all boxes in current frame
EExpand all boxes in current frame by 10%
Del / BackspaceDelete selected box
EscDeselect active drone / selected box

Memorise mode

MToggle memorise mode on / off

With memorise on, clicking the canvas places a box sized to match the largest box in the last frame that has any annotation, centred on your click. Useful for quickly filling annotation gaps when a drone is continuously visible.

File operations

SaveCommit all changes to coordinates.txt (creates .bak on first save)
Revert frameReset current frame to original GT
Revert SeqDiscard all unsaved changes and restore entire sequence to original GT
Expand AllExpand annotations in current frame and all subsequent frames by 10%
ReloadReload all annotations from disk
